# 银行流水识别系统 (LSSB)

银行流水图片智能识别系统，将银行流水图片自动转换为结构化的 JSON 数据。

## 项目架构

```
流水识别/
├── LLM-Deployment/      # 部署相关（Dockerfile、启动脚本、部署说明文档）
├── LLM-Service/          # 服务代码（FastAPI 流水识别服务，多版本迭代）
├── LLM-Test/             # 测试代码（准确率测试、单图测试、URL 测试）
└── LLM-Training/         # 训练代码（模型微调相关）
```

## 版本历史

| 版本 | 说明 | 核心变化 |
|------|------|----------|
| v0.4 | 初始版本 | 布局解析 + LLM 分离模式 |
| v0.5 | 优化版 | 流程优化，效果提升 |
| v0.6 | 多行并行 | 多行并行处理优化 |
| v0.7 | VL 直抽 | 直接使用 Qwen VL 抽取，无需布局解析流程 |

## 核心功能

1. **图片解析**：接收 base64 编码的银行流水图片，解析为结构化数据
2. **智能识别**：使用大语言模型/视觉语言模型提取关键字段
3. **数据规范化**：统一格式，提取交易日期、流水号、对方户名、借方/贷方金额等

## 技术架构

- **Web 框架**：FastAPI，提供 RESTful API 接口
- **外部服务**：布局解析 API、LLM/VL API
- **数据处理**：HTML 解析、Markdown 转换、JSON 后处理
- **异步处理**：使用线程池处理耗时操作

## 快速开始

### 安装依赖

```bash
cd LLM-Service
pip install -r requirements.txt
```

### 启动服务

```bash
cd LLM-Service
uvicorn app:app --reload --host 0.0.0.0 --port 8000
```

### Docker 部署

```bash
# 构建镜像
docker build -f LLM-Deployment/Dockerfile -t liushui:0.7 .

# 运行容器
docker run -d -p 28765:8000 liushui:0.7
```

## 使用示例

```bash
curl -X POST http://localhost:8000/layout-parsing \
  -H "Content-Type: application/json" \
  -d '{"image": "base64编码的图片数据"}'
```

## 配置说明

编辑 `LLM-Service/config.yaml`：

```yaml
layout_api_url: "http://172.16.21.190:18188/layout-parsing"  # 布局解析 API
llm_api_url: "http://172.16.21.190:8060/v1/chat/completions"  # LLM API
model_name: "qwen3_4B"  # 模型名称
```
